新版本

为每个单独的工作表都建立了单独的 data 对象，所以应该现在 instance.datas 中找到对应的 data 对象，在进行操作


// instance 你创建的实例，公有方法可以直接调用，私有方法不可以

const target_sheet_name = `sheet2`

const target_data = instance.datas.find(it => it.name === target_sheet_name)
