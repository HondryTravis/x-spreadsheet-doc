# 如何书写 md 文档

# 标题

使用#标记 作为开头，中间隔开空格， 并且对应 h1-h6

注意事项： 搜索内容仅对一至三级标题(h1-h3)生效

# 一级标题

有且只有一个

  # 文字
1

# 二级标题

  ## 文字
1

# 三级标题

  ### 文字
1

# 四级标题

  #### 文字
1

# 五级标题

  ##### 文字
1

# 六级标题

  ###### 文字
1

# 加粗、斜体

你可以使用 * 或者 _ 作为标记开头，来对文字加粗或者斜体

# 加粗

使用 ** 或者 __ 来标记，成对出现

例子

**你的文字** : 你的文字

__你的文字__ : 你的文字

# 斜体

使用 * 或者 _ 来标记， 成对出现

例子

*你的文字* : 你的文字

_你的文字_ : 你的文字

# 加粗斜体

使用*** 或者 ___ 作为标记，成对出现

例子

***你的文字*** : 你的文字

___你的文字___ : 你的文字

# 列表

# 有序列表

使用 1. 来标记，英文标点

这是一个有序列表 `1. 测试`

  1. 测试
  2. 测试

# 无序列表

可以使用 -、+、* 来写无序列表

例子 `- 测试`, `+`、`*`都相同

  • 测试

  • 测试

# 引用

使用 > 标记作为开头可以嵌套

例子

这是一段引用1

这是一段引用2

这是一段引用3

引用可以在列表中使用

# 超链接

在页面加入超链接 [文字](地址) 或者直接是地址 <地址>

例子

[文字](地址): 百度

<地址>: https://www.baidu.com

# 图片

在页面加入图片,使用 ![文字描述](地址)

例子
